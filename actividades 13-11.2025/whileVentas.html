Enunciado del ejercicio: Registro de Ventas Diarias Desarrollar un programa en JavaScript que permita registrar las ventas diarias de un vendedor y evaluar su rendimiento según el promedio de ventas. Descripción: El programa debe permitir al usuario ingresar, de forma repetida, el monto de ventas realizadas cada día. Cada ingreso corresponde a las ventas de un día. El programa finaliza cuando el usuario ingresa 0. No se permiten valores negativos; si el usuario ingresa un número negativo, se debe mostrar una alerta indicando que elvvalor no es válido. Al finalizar, el programa debe: -Calcular y mostrar el promedio de ventas por día. -Evaluar el rendimiento del vendedor según el promedio: -Promedio mayor a 1000 → "Rendimiento excelente" -Promedio entre 500 y 1000 → "Rendimiento bueno" -Promedio menor a 500 → "Rendimiento bajo"
